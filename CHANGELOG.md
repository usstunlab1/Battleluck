# Changelog

## Unreleased

### Added

- Adaptive event combat-drill catalog and bounded event-start NPC planning.
- Admin solo-practice modes for follow, fight, mirror, status, and cleanup.
- Short `.ai` aliases for requests and practice commands.
- Professional administration, architecture, development, and contributor documentation.

### Changed

- Event console names support up to 32 characters.
- Documentation is aligned with server-side V Rising BepInEx/IL2CPP and VCF practices.

### Limitations

- Mirror behavior is close native-NPC following, not client-input replication.
- Quest data and canonical world-progression mapping are not implemented.
